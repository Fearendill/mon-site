## 📋 À propos

Site web personnel entièrement développé from scratch en **HTML / CSS / JavaScript vanilla**, sans framework.  
Disponible en **deux langues** 🇫🇷 🇬🇧 pour toucher un public plus large.

---

## 🗂️ Structure du projet

```
mon-site/
├── 📄 index.html          # Page principale (Français)
├── 📄 index_en.html       # Page principale (English)
├── 🎨 style.css           # Styles globaux
├── ⚙️  app.js              # Logique & interactivité
├── 📁 assets/             # Images, icônes, médias
├── 🗺️  sitemap.xml         # Plan du site (SEO)
├── 🤖 robots.txt          # Directives robots (SEO)
└── 🔍 google[...].html    # Vérification Google Search Console
```

---

## ✨ Fonctionnalités

- 🌍 **Bilingue** — Français & Anglais
- 📱 **Responsive** — Compatible mobile, tablette et desktop
- ⚡ **Léger** — Zéro dépendance, chargement rapide
- 🔎 **SEO-ready** — Sitemap XML + robots.txt + Google Search Console

---

## 🚀 Lancer le projet en local

```bash
# Cloner le dépôt
git clone https://github.com/Fearendill/mon-site.git

# Ouvrir dans le navigateur
cd mon-site
open index.html
```

> Aucune installation requise — fonctionne directement dans le navigateur.
